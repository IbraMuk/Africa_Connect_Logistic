# Images du projet Africa Connect Logistic

## Logo principal

Pour intégrer le logo de l'entreprise, placez votre fichier image ici avec le nom :

- **Nom du fichier** : `logo.png` (recommandé)
- **Formats acceptés** : `.png`, `.jpg`, `.jpeg`, `.svg`
- **Taille recommandée** : 500x500px minimum pour une bonne qualité
- **Fond** : Transparent (PNG) ou blanc

## Alternative

Si vous préférez utiliser un autre nom de fichier, modifiez le composant `Logo.tsx` dans `frontend/src/components/Logo.tsx`.
